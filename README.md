# Descrição

API REST que irá gerir as infra-estruturas. A API deve suportar fazer um CRUD completo das mesmas.
sendo que, dado a necessidade e finalidade da api, não há camada de persistência então devemos efectuar 
o registo dos dados. 

## Executar a api
```bash
$ docker-compose up
```
